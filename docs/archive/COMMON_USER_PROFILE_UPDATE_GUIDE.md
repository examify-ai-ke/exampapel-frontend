# Common User Profile Update Guide

## 🎯 How Common Users Update Their Profile

### What is a "Common User"?
A common user is a regular student or user who:
- Has signed up for an account
- Is NOT an admin or manager
- Wants to update their profile information
- Should NOT access the admin dashboard

## 📍 Where to Update Profile

### New Way (Recommended)
```
URL: http://localhost:3000/profile
```

### Old Way (Still Works)
```
URL: http://localhost:3000/dashboard/profile
```

## 🚀 Step-by-Step Guide

### Step 1: Log In
1. Go to the homepage
2. Click "Sign In" button
3. Enter your email and password
4. Click "Sign In"

### Step 2: Access Your Profile
**Option A: From Header (Recommended)**
1. Look at the top-right corner
2. Click your avatar (profile picture)
3. A dropdown menu appears
4. Click "Profile"
5. You're now on your profile page

**Option B: Direct URL**
1. Type in address bar: `http://localhost:3000/profile`
2. Press Enter
3. You're now on your profile page

### Step 3: Edit Your Profile
1. Click "Edit Profile" button
2. Make changes to any field:
   - First Name
   - Last Name
   - Email
   - Phone Number
   - Address
   - State/Province
   - Country
3. Changes auto-save automatically (2 seconds after you stop typing)
4. Watch for "Auto-saved" indicator
5. Click "Done" when finished

### Step 4: Change Password (Optional)
1. In Account Settings section, click "Change" button
2. Enter your current password
3. Enter your new password (minimum 8 characters)
4. Confirm your new password
5. Click "Change Password"
6. Success message appears

## 💡 Tips & Tricks

### Tip 1: Auto-Save
- You don't need to click a save button
- Changes save automatically 2 seconds after you stop typing
- Watch for the "Auto-saved" indicator

### Tip 2: Password Visibility
- Click the eye icon to show/hide your password
- Useful for verifying you typed it correctly
- Each password field has its own toggle

### Tip 3: Mobile Access
- Profile page works on mobile devices
- Tap your avatar in the header
- Tap "Profile"
- Same features as desktop

### Tip 4: Quick Updates
- You can update your profile from any page
- Just click your avatar and select "Profile"
- No need to go through dashboard

## 🐛 Troubleshooting

### Problem: Can't access profile page
**Solution:**
- Make sure you're logged in
- Check if you're at the correct URL: `/profile`
- Try logging out and logging back in

### Problem: Changes not saving
**Solution:**
- Check your internet connection
- Wait for "Auto-saved" indicator
- Refresh the page to verify changes
- Check browser console for errors

### Problem: Password change fails
**Solution:**
- Verify current password is correct
- Check new password is at least 8 characters
- Verify passwords match
- Try again

### Problem: Can't see password visibility toggle
**Solution:**
- Refresh the page
- Clear browser cache
- Try a different browser
- Check if JavaScript is enabled
