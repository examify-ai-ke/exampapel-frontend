API Endpoints Added
All endpoints require admin role:

GET /api/v1/audit-logs/audit - Query audit logs with filters
GET /api/v1/audit-logs/activity - Query activity logs with pagination
GET /api/v1/audit-logs/audit/stats - Audit log statistics
GET /api/v1/audit-logs/activity/stats - Activity log statistics
GET /api/v1/audit-logs/user/{id}/activity - User activity summary
GET /api/v1/audit-logs/security/suspicious - Suspicious activity detection
